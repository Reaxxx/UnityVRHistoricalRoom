# UnityVRHistoricalRoom

Ottoman Imperial Treasury: Museum Experience
🏛 Project Overview
This project is a high-fidelity museum experience developed using Unity 6 (URP). It offers an immersive atmosphere that blends 19th-century Ottoman palace aesthetics with Victorian-era furniture and historical artworks. The experience is designed to authentically showcase Ottoman cultural heritage in a digital environment.

🚀 Key Features
Advanced Visual Quality: Built using Unity 6 and the Universal Render Pipeline (URP) for realistic lighting and material details.

Custom 3D Models: Enriched with period-specific objects such as an Imperial Tughra and a traditional Astrolabe.

Atmospheric Depth: A "lived-in" ambiance created with dust motes and golden flecks drifting within light shafts (God Rays).

Historical Art Gallery: A high-resolution digital exhibition consisting of portraits of Ottoman Sultans and historical calligraphy.

🛠 Technical Details
Engine: Unity 6 (6000.0.40f1)

Render Pipeline: URP (Universal Render Pipeline)

Target Platform: PC / VR

📜 Installation
Clone the repository: git clone https://github.com/Reaxxx/UnityVRHistoricalRoom

Open the project in Unity Hub using version 6 (6000.0.40f1).

Ensure the Universal Render Pipeline package is correctly configured in Project Settings.
